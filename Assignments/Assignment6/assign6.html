This Tableau dashboard features maps and a line chart depicting data sourced from the National Agricultural Statistics Service (NASS) within the
USDA. The first map shows honey production averages with the average production value listed as additional information within the tooltip. The
second map shows the average yield per colony in each state and displayed with 5 breaks. The tooltip for this map also includes the average number
of active honey producing colonies. The line chart shows how the price of honey has changed over the period from 1998 to 2012, which is the same
period the averages are generated from. The use of averages was important for this dataset as there was not information available for every state
each year. I also included an image of honey sourced from BeeCulture.com .
